SD card backup image

When the magnetometer arrived from Dr Steve Marple of Lancaster University, the image on the Raspberry Pi image had not been expanded, so that we could make up a backup of the initial configuration.

On 18 November 2013, Ciaran O'Shea expanded the image by loggin in via ssh as pi and running
sudo raspi-config
followed by selecting Expand Filesystem, and then Finish followed by rebooting the Pi.

This unexpectedly had the effect of regenerating the ssh keys for the Pi.

Testing data connexion

Ciaran did all of the above using the school network, thus confirming that the network point in L5 which we might use for the raspberry pi works.
